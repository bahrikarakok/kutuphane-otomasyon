Bu proje, C# ve MS SQL Server kullanılarak geliştirilmiş bir kütüphane yönetim otomasyonudur. Temel CRUD (Ekleme, Listeleme, Güncelleme, Silme) işlemlerini ve veritabanı yönetimini içermektedir.

🛠️ Kullanılan Teknolojiler
Dil: C#

Platform: .NET Desktop (Windows Forms)

Veritabanı: Microsoft SQL Server

IDE: Visual Studio 2019

✨ Temel Fonksiyonlar
Üye Yönetimi: Kütüphane üyelerini kaydetme ve bilgilerini güncelleme.

Kitap Yönetimi: Kitap envanterini tutma ve kategorilendirme.

Emanet İşlemleri: Ödünç verilen kitapların takibi ve iade süreçleri.

⚙️ Kurulum Notu
Projenin çalışması için yerel SQL Server'ınızda veritabanının kurulu olması ve App.config dosyasındaki (veya kod içindeki) Connection String ayarlarının kendi sunucu adınıza göre güncellenmiş olması gerekmektedir.
